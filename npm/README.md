# @acp-protocol/cli

AI Context Protocol CLI - Token-efficient code documentation for AI systems.

## Installation

```bash
npm install -g @acp-protocol/cli
```

## Usage

```bash
# Initialize a project
acp init

# Index your codebase
acp index

# Query the cache
acp query file src/main.rs
acp query symbol MyFunction

# Generate a primer for AI context
acp primer
```

## Alternative Installation Methods

```bash
# Via Cargo (Rust)
cargo install acp

# Via Homebrew (macOS/Linux)
brew install acp-protocol/tap/acp
```
